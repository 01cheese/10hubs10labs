## 📦 How to Use

Each project is written in markdown and automatically rendered. You can browse, learn, and try any project directly from the site.

To run locally:

```bash
git clone https://github.com/01cheese/10hubs10labs.git
cd 10labs10habs
npm install
npm run dev
```

Open `http://localhost:4321` in your browser.
